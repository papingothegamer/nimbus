4. **Action Wiring**

5. **Bug Fixes: Engine Stability & State Transfers**
   - **Vector Out-Of-Range Crash:** Fixed a thread contention crash during track deletion by placing `processLock` scopes inside `AudioGraph::getLatencySamples` and safely cleaning up `trackNodes` mapping before `Mixer::removeTrack`.
   - **PDC Track Duplication Desync:** Fixed 0-latency reports by calling `prepareToPlay` on newly duplicated plugins prior to graph insertion, preventing duplicate tracks from slipping out of sync.
   - **Track Solo PDC Stalling:** Found that soloing a track caused muted tracks to completely skip `processBlock`. This stalled their internal plugin delay lines, making them out of sync when un-soloed. Fixed by adding a `silencedBySolo_` flag to `Track`, forcing internal processing to continue silently.
   - **Stock Plugin Serialization:** Created state persistence architecture (`getStateInformation` / `setStateInformation`) inside `IStockPlugin` and fully implemented it for all 7 stock plugins (CloudEQ, Delay, Chorus, Compressor, Filter, Gain, Reverb). Wired this up in `NimbusEngine::duplicateTrack` so all parameters now carry over perfectly when duplicating a track.

6. **Bug Fixes: Track Grouping UI**
   - Fixed group track header name not rendering correctly.
   - Refactored `TrackHeaderComponent` and `GroupTrackHeaderComponent` layout calculations to use absolute anchoring for right-aligned UI elements (solo, mute, arm buttons). This prevents buttons from disappearing when the track panel is sized narrowly due to UI flex layout truncation.
   - Diagnosed an issue where duplicating a track within a group causes the duplicated track's clip to become invisible in the timeline, which appears to be unrelated to `AudioThumbnailCache` or `UniqueFileInputSource`.
